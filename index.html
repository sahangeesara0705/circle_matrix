circle-matrix

Initialize circle-matrix

Creates an instance bound to a container element and draws it for the first time. Every other method below assumes you've already got an instance from circleMatrix.init().

    
const circleMatrixInstance1 = circleMatrix.init(
    document.getElementById("circle-matrix"),
    {
        onColor: "blue",
        circleSize: 8
    }
);
circleMatrixInstance1.render();

Using .changeOne()

Turns a single cell on or off by row (i) and column (j) index. Doesn't redraw by itself — call .render() after to see the change.


circleMatrixInstance2.changeOne(1, 1, 1);
circleMatrixInstance2.render();

Using .changeRow() and .changeColumn()

Sets an entire row or column to the same state in one call — handy for building crosshair or grid-line effects.


circleMatrixInstance3.changeRow(1, 1);
circleMatrixInstance3.changeColumn(1, 1);
circleMatrixInstance3.render();

Using .changeMatrix()

Replaces the whole grid at once from a 2D array — the array just needs to match the instance's width/height. This is how circlematrix_11x11.js glyphs get drawn: look one up with getCircleMatrixGlyph() and pass it straight in.


circleMatrixInstance4.changeMatrix(getCircleMatrixGlyph("C"));
circleMatrixInstance4.render();

Using .changeAll()

Sets every cell in the grid to the same state — the quickest way to blank a grid (0) or light it up entirely (1).


circleMatrixInstance5.changeAll(1);
circleMatrixInstance5.render();

Using .getState()

Returns a copy of the current grid as a plain 2D array of 0s and 1s — useful for saving a pattern, comparing states, or debugging. It's a copy, so mutating the result won't affect the instance.


circleMatrixInstance6.changeOne(0, 0, 1);
circleMatrixInstance6.changeOne(10, 10, 1);
circleMatrixInstance6.render();
console.log(circleMatrixInstance6.getState());
